About This Project

The Outlier is an open-source NBA analytics tool designed to spotlight the most unexpected and statistically significant game performances. Unlike typical stat summaries or highlight reels, this project uses a custom scoring system to identify which players and teams most deviated—positively or negatively—from their season averages.

🎯 Purpose

  • Provide fans and analysts with deeper insight into standout performances.
  • Use data-driven logic to highlight both underrated contributions and shocking underperformances.
  • Serve as a transparent, educational alternative to proprietary tools like Cleaning the Glass or BBall Index.

⚙️ How It Works

Game data is collected using the nba_api Python package. Each stat is compared to season-long averages at the player and team levels. A blended outlier score is calculated using both percent difference and raw volume increase, ensuring that context matters (e.g., scoring 36 when you average 23 is more impressive than scoring 16 when you average 8).
